structural pseudo classes

  1. before pseudo-class: Defines a style before the element is displayed
  2. after pseudo-class: Defines a style after the element is displayed
  3. first-child pseudo-class: Selects the first child of a specified element
  4. last-child pseudo-class: Selects the last child of a specified element
  5. nth-child pseudo-class: Selects every nth child of a specified element
  6. nth-of-type pseudo-class: Selects every nth child of a specified element of the same type
  7. nth-last-of-type pseudo-class: Selects every nth child of a specified element of the same type, starting from the last child
  8. first-of-type pseudo-class: Selects the first child of a specified element of the same type
